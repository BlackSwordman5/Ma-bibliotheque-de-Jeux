Akinofa

Développeur : Pixel Lantern

Éditeur : Pixel Lantern

Date de Sortie : 16 Juillet 2021

Genre : Shoot'em Up, Roguelike

Synopsis : Akinofa est un shoot'em up arcade avec des éléments de roguelite. Aux commandes d'un étrange poisson mort-vivant, éliminez vos ennemis dans les profondeurs et tentez d'aller le plus loin possible pour grimper dans les classements mondiaux.
